## Multi-service add-on playbook

Three layers: (1) one anchor program owns the care bundle, (2) optional medication add-on for the second Rx lane on the same Stripe subscription, (3) peptides as separate SKUs with member 20% off — never duplicate the care bundle.

| IV membership | ELEVATED IV $199 | Non-Rx · no hormone/GLP combo SKUs | 20% off à la carte IV + peptides | Separate from Rx combo — can stack with layer 3 peptides |

### Enrollment steps

1. $79 Wellness Assessment + baseline labs ($199 hormones · $299 if GLP-1 in mix)
2. Provider review — document all lanes: weight, hormones, recovery, vitality, metabolic adjuncts
3. Pick anchor = primary clinical goal (weight → GLP-1 anchor; hormones → TRT/HRT anchor)
4. If second Rx lane indicated → select medication add-on in Elevated Combo Selector (one subscription, two line items)
5. Send combo checkout link OR update existing subscription via update-subscription-addon
6. Peptides (layer 3): after consent + gates → separate Send Payment Link per SKU (member −20% each line)
7. Quote total monthly = combo total + sum of peptide lines (show math on paper)
8. Chart: elevated_program (anchor) + elevated_program_addon + peptide SKUs in care plan
9. Staging OK: physician may start anchor only; add medication add-on after 8–12 weeks stable
